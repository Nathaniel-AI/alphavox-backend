Algorithm Implementation

The Enhanced Porter Stemming Algorithm employed in this research adheres to the canonical Porter algorithm while introducing special considerations for neurodivergent communication patterns.

Step 1: Word Normalization

Words are normalized and prepared for stemming through case standardization and special character handling, with specific attention to patterns common in neurodivergent writing.

"Connecting" → normalize → "connecting"
"HAPPINESS" → normalize → "happiness"
Step 2: Suffix Removal (Step 1a/1b/1c)

This stage applies complex linguistic rules to manage plurals and common verb conjugations. This is particularly important for neurodivergent analysis as it standardizes different variations of the same base concept.

"connections" → "connection" (1a)
"connected" → "connect" (1b)
"study" → "studi" (1c)
Step 3: Derivational Suffix Handling (Steps 2-4)

These steps handle derived word forms with consistent application of rules based on syllable count and word structure.

"conditional" → "condit" (Step 2)
"activate" → "activ" (Step 3)
"formalize" → "formal" (Step 4)
Step 4: Special Case Handling for Neurodivergent Communication

This research introduces novel refinements to handle special cases common in neurodivergent communication, including repetition patterns, filler words, and specialized vocabulary.

Identify filler patterns: "um, like, actually"
Process repetition: "very very very" → repetition pattern with intensity marker
Step 5: Stem Consolidation

The final step unifies stems and calculates compression metrics valuable for research analysis.

Word count: 150 → Unique stems: 65
Stem reduction ratio: 56.7%

Code Implementation Sample

def _step1a(self, word: str) -> str: """ Apply step 1a of the Porter stemming algorithm Args: word: The word to stem Returns: The stemmed word """ for suffix, replacement in self.step1a_suffixes: if self._ends_with(word, suffix): return self._replace_suffix(word, suffix, replacement) return word def _step1b(self, word: str, m: int) -> str: """ Apply step 1b of the Porter stemming algorithm Args: word: The word to stem m: The measure of the word Returns: The stemmed word """ if self._ends_with(word, 'eed'): if m > 0: return self._replace_suffix(word, 'eed', 'ee') return word for suffix, replacement in [('ed', ''), ('ing', '')]: if self._ends_with(word, suffix): preceding = self._replace_suffix(word, suffix, '') if self._contains_vowel(preceding): word = preceding if self._ends_with(word, 'at'): return word + 'e' if self._ends_with(word, 'bl'): return word + 'e' if self._ends_with(word, 'iz'): return word + 'e' if self._ends_double_consonant(word) and not word.endswith(('l', 's', 'z')): return word[:-1] if m == 1 and self._ends_cvc(word): return word + 'e' return word return word
