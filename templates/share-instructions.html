How to Find the Share Button

First, make sure you're in the right view to see the Share button!

Step 1: Exit Full Screen Mode

Press the Esc key on your keyboard to exit full screen mode

After pressing Esc, you should see the Replit editor interface with:

  • Files panel on the left
  • Code editor in the middle
  • Preview window on the right

Still Can't Find It?

Make sure you:

  1. Are not in full screen mode (press Esc if you are)
  2. Can see the Replit editor interface (with files on the left)
  3. Look at the very top of the window, not inside the web application
